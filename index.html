  • Personal Project: Ham Grid

    Amateur radio enthusiasts use something called the Maidenhead Grid System to report their position. After learning about this I decided it would be cool to overlay that grid on a google map and to plot the user's position using GPS. So I created a PWA called Ham Grid to do just that.

    Technologies Used: HTML, CSS, JavaScript, Google Maps, React, Service Workers

  • Personal Project: Scribblez

    I had the realization one day that every time a new technology came out I wanted to learn I'd have to invent a new sample app to build. That seemed somewhat wasteful so I set out to create a generic project I could reuse for each new technology. I ended up creating the idea of a social network similar to twitter. I call it Scribblez. Scribblez currently consists of an SQL database (schema and sample data) and a REST API server.

    Technologies Used: Node, Hapi, Knex, PostgreSQL
